# The Weekly Threat Brief: Copilot Exploits, Industrial Controller Failures, and Supply Chain Chaos

**TL;DR**
*   **Copilot Compromised:** A critical "SearchLeak" vulnerability chain allows 1-click data theft from M365 environments.
*   **Industrial Exposure:** A massive batch of vulnerabilities in Wertheim SafeController software exposes physical security infrastructure to remote takeover.
*   **Supply Chain & CMS:** WordPress plugins continue to be a primary vector for RCE and privilege escalation, while unmaintained software like Responsive FileManager is actively being weaponized.
*   **Data Breach:** 137,000 school staff accounts were compromised via a Salesforce-based attack on the Infinite Campus system.

---

## Critical Vulnerabilities & Exploits

### Microsoft 365 Copilot "SearchLeak" [BleepingComputer]
A critical vulnerability chain has been identified that turns Microsoft 365 Copilot into a data exfiltration tool. By leveraging a specially crafted URL, an attacker can bypass standard access controls to pull sensitive data from a user’s mailbox, OneDrive, or SharePoint.

*Phalanx Thoughts: This is the nightmare scenario for GenAI adoption. When the tool designed to "assist" productivity becomes an automated exfiltration engine, the traditional perimeter is effectively bypassed. Audit your Copilot permissions immediately and restrict external link processing.*

### Wertheim SafeController Series [CVE]
A staggering series of vulnerabilities (CVE-2026-34022 through CVE-2026-34030) has been disclosed for Wertheim SafeController software. These range from hard-coded cryptographic keys and unauthenticated direct access to web data, to RCE and path traversal.

*Phalanx Thoughts: This is a catastrophic failure of secure-by-design principles for physical security hardware. If you manage these controllers, assume they are already compromised if they have been exposed to any network segment.*

### Responsive FileManager RCE [CVE]
CVE-2026-5482 highlights the danger of "abandonware." This unmaintained file manager allows unauthenticated attackers to upload arbitrary files via the `dialog.php` endpoint, leading to full Remote Code Execution.

### Foxit AI PDF JavaScript Execution [CVE]
CVE-2026-12057 allows remote code execution by failing to properly sandbox JavaScript embedded within PDF files.

The WordPress ecosystem continues to be a high-frequency target for attackers looking for quick entry points.

*   **WooCommerce PDF Invoice Builder (CVE-2026-52704):** Critical RCE vulnerability. Patch immediately.
*   **Masteriyo LMS (CVE-2026-49111):** Privilege escalation vulnerability allowing unauthorized users to gain elevated access.
*   **Faust.js (CVE-2026-49062):** Authentication bypass via password recovery exploitation.
*   **MasterStudy LMS Pro (CVE-2025-64215):** Broken access control.
*   **GetPaid (CVE-2026-49064):** Sensitive data exposure.
*   **Really Simple SSL (CVE-2026-48969):** Subscriber-level broken access control.

## Malware & Data Breaches

### Infinite Campus Breach [BleepingComputer]
The ShinyHunters extortion gang has struck again, targeting the Infinite Campus K-12 student information system. By exploiting a Salesforce integration, they successfully exfiltrated personal data belonging to 137,000 school staff members.

### REDCap Server Espionage [BleepingComputer]
Chinese-linked threat actors are actively targeting exposed REDCap (Research Electronic Data Capture) servers. The campaign utilizes "InfiniteRed" malware to exfiltrate sensitive medical research data.

*Phalanx Thoughts: Research institutions are often the "soft underbelly" of the cybersecurity world. If you are hosting sensitive research data, your REDCap instances must be isolated from the public internet and monitored for anomalous egress traffic.*

## Defender Takeaways

1.  **Audit AI Integrations:** Review your Microsoft 365 Copilot configuration. Ensure that "SearchLeak" mitigation strategies (such as strict URL filtering and data sensitivity labeling) are enforced before allowing broad access to sensitive SharePoint/OneDrive repositories.
2.  **Inventory "Dead" Software:** The Responsive FileManager exploit is a reminder that unmaintained software is a ticking time bomb. Scan your web-facing assets for abandoned plugins or libraries and replace them with supported alternatives.
3.  **Harden Industrial Controllers:** If you utilize Wertheim SafeController or similar industrial/physical security hardware, verify they are not internet-facing. If they must be networked, place them behind a strict jump box with multi-factor authentication and monitor all traffic for unauthorized WebSocket or API calls.
4.  **Prioritize CMS Patching:** With the high volume of WordPress-related vulnerabilities this week, shift your patching cycle to prioritize RCE and Privilege Escalation bugs in your plugin stack. If a plugin is no longer receiving updates, it is a liability—remove it.
